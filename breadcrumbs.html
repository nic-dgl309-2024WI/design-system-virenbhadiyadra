07 Breadcrumbs

Example

Using the combination of div, anchor links, and span to style amazing breadcrumbs.

Home
Home / Current Page
Home / Category / Current Page
HTML
<div class="breadcrumb-links">
<a href="#">Home</a>
</div>

<div class="breadcrumb-links">
<a href="#">Home</a>
<span class="separator">/</span>
<span class="current-page">Current Page</span>
</div>

<div class="breadcrumb-links">
<a href="#">Home</a>
<span class="separator">/</span>
<a href="#">Category</a>
<span class="separator">/</span>
<span class="current-page">Current Page</span>
</div>

Link separators

Link separators can be changed by simply changing content of span element from " / " to your desire symbol.

Home
Home > Current Page
Home > Category > Current Page
HTML
<div class="breadcrumb-links">
<a href="#">Home</a>
</div>

<div class="breadcrumb-links">
<a href="#">Home</a>
<span class="separator"> > </span>
<span class="current-page">Current Page</span>
</div>

<div class="breadcrumb-links">
<a href="#">Home</a>
<span class="separator"> > </span>
<a href="#">Category</a>
<span class="separator"> > </span>
<span class="current-page">Current Page</span>
</div>
