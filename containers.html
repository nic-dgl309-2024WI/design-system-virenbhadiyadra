04 Containers

The table below will show you how each container size works.

Container class X-Small <570px Small ≥570px Medium ≥750px Large ≥980px X-Large ≥1150px XX-Large ≥1370px
.container 100% 530px 700px 950px 1100px 1300px
.container-sm 100% 530px 700px 950px 1100px 1300px
.container-md 100% 100% 700px 950px 1100px 1300px
.container-lg 100% 100% 100% 950px 1100px 1300px
.container-xl 100% 100% 100% 100% 1100px 1300px
.container-xxl 100% 100% 100% 100% 100% 1300px
.container-fluid 100% 100% 100% 100% 100% 100%

Containers Example

HTML
<div class="container">I am the default container class, my max-width will change on different screen sizes.</div>
<div class="container-sm">I will set the width to 100% for screen sizes up to <b>570 pixels</b>.</div>
<div class="container-md">I will set the width to 100% for screen sizes up to <b>750 pixels</b>.</div>
<div class="container-lg">I will set the width to 100% for screen sizes up to <b>980 pixels</b>.</div>
<div class="container-xl">I will set the width to 100% for screen sizes up to <b>1150 pixels</b>.</div>
<div class="container-xxl">I will set the width to 100% for screen sizes up to <b>1370 pixels</b>.</div>
