Containers Example

HTML <div class="container">default container class, mt max-width will change on different screen size.</div>

<div class="container-sm">set the width to 100% for screen sizes up to 570 pixels.</div>

<div class="container-md">set the width to 100% for screen sizes up to 750 pixels.</div>

<div class="container-lg">set the width to 100% for screen sizes up to 980 pixels.</div>

<div class="container-xl">set the width to 100% for screen sizes up to 1150 pixels.</div>

<div class="container-xxl">set the width to 100% for screen sizes up to 1370 pixels.</div>
